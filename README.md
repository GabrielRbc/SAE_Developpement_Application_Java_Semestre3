# Application d’entraînement au secourisme (SST)

## Description
Ce projet consiste à développer une application ludique destinée aux personnels de l'IUT qui sont **Sauveteurs Secouristes du Travail (SST)**.  
Elle permet de réviser régulièrement le **plan d’intervention SST (PREFAS : Protéger, Examiner, Faire Alerter, Secourir)** au travers de différents modes de jeu :  
- Révision de la grille des pictogrammes.  
- QCM interactifs.  
- Scénarios fixes et scénarios aléatoires pas à pas.

L’objectif est d’aider les SST à entretenir leurs compétences grâce à un outil numérique simple et pédagogique.
